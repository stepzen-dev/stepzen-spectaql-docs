Queries

weatherReport

Returns a WeatherReport

Name Description
latitude - Float!
longitude - Float!

Example

Query
query weatherReport($latitude: Float!, $longitude: Float!) {
  weatherReport(latitude: $latitude, longitude: $longitude) {
    date
    description
    feelsLike
    latitude
    longitude
    temp
    units
  }
}
Variables
{"latitude": 123.45, "longitude": 987.65}

The Float scalar type represents signed double-precision fractional values as specified by IEEE 754.

Types

PageInfo

PageInfo indicates if more results are available in a connection. See GraphQL Cursor Connections Specification

Field Name Description
endCursor - String! Cursor corresponding to the last node in edges
hasNextPage - Boolean! Indicates whether more edges exist following the set defined by the pagination arguments.
hasPreviousPage - Boolean! Indicates whether more edges exist prior to the set defined by the pagination arguments.
startCursor - String! Cursor corresponding to the first node in edges
Example
{
  "endCursor": "abc123",
  "hasNextPage": true,
  "hasPreviousPage": false,
  "startCursor": "abc123"
}
